# Personas (MVP)

## P-01 YouTube初心者（Primary）

- **概要**: 動画編集経験がほぼないが、解説動画を公開したい個人ユーザー
- **ゴール**: テーマ入力から短時間で動画を作成し、投稿可能なファイルを得る
- **課題**:
  - 編集ソフトの学習コストが高い
  - 台本と演出を同時に作るのが難しい
  - 動画書き出し手順でつまずきやすい
- **期待価値**:
  - ブラウザのみで完結
  - YAML編集で最小限の調整が可能
  - WebMを確実にダウンロードできる

## ペルソナ適用方針

- 本フェーズではユーザー回答に基づき、MVP最優先の1ペルソナに集中する
- 将来フェーズで教育系クリエイター等の追加ペルソナを拡張する
